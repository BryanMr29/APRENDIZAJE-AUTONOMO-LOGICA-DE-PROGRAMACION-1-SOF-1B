# JUEGO DE PIEDRA PAPEL O TIJERAS
## OBJETIVO DEL PROGRAMA
#### Crear un juego de piedra papel o tijeras que permita al usuario competir contra la computadora
## PRINCIPALES FUNCIONALIDADES DEL CODIGO
#### El juefo de piedras papel o tijeras se implementa a travez de las siguientes funciones:
### logica_de_juego
#### Determina el ganador del juego segun la opcion escogida por el usuario y la computadora y devuelve el resultado del juego ademas de tener un contador que suma puntos por cada victoria
### mostrar_puntos
#### Muestra el total de puntos acumulados por el usuario, computadora o empates
### LIBRERIA RANDOM
#### El juego usa la libreria random para la eleccion aleatoria de la computadora ¿
## LOGICA DEL JUEGO
#### El usuario debe ingresar su nombre
#### El usuario debe elegir una de las tres opciones piedra, papel o tijeras
#### La computadora hace una eleccion aleatoria
#### Se comparan las opciones y se elige un ganador
#### Se actualizan los puntos del jugador y la computadora
#### Se muestran el resultado del juego y los puntos actuales de cada jugador
#### Se pregunta al usuario si desea seguir jugando o salir (el codigo se ejecutara indefinidamente hasta que el usuario decida ya no jugar)
#### Se muestra el ganador y un mensaje de despedida
